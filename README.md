## Install

	$ npm install

## Development

    npm run dev chrome
    npm run dev firefox
    npm run dev opera
    npm run dev edge

## Build

    npm run build chrome
    npm run build firefox
    npm run build opera
    npm run build edge

## Testing

    npm test

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
