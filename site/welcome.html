Everything that's new

Here's what we've been working on. Every feature was the result of user feedback.

Command Palette

Click the extension icon for an instant searchable list of all your shortcuts. Navigate with arrows, trigger with Enter.

Shortcut Packs

One-click install curated collections — Vim, Emacs, YouTube, Tab Manager, and more. Customize or remove anytime.

Cloud Sync

Shortcuts automatically sync across all your Chrome instances via your Google account. No extra account needed.

Shortcut Groups

Organize into collapsible groups. Enable/disable, rename, or delete entire groups at once. Packs import as groups.

Shortcut Recorder

Click Record and press your keys. Supports multi-key sequences like Vim's g g. No more guessing key names.

Conflict Detection

Warns when your shortcuts conflict with browser defaults or duplicates. Platform-aware — knows Mac vs Windows shortcuts.

Plus 40+ more improvements

Reader mode, dark mode, QR codes, picture-in-picture, video speed controls, tab sorting, search YouTube/Wikipedia/GitHub, and much more.

Dark mode

Settings page follows system preference, plus a one-key dark mode action for any website.

Share via link

Generate a shareable URL for any shortcut or group. Anyone with Shortkeys can import it with one click.

Live reload

Save your shortcuts and they take effect immediately in all open tabs. No more refreshing every page.

Firefox support

Shortkeys now works on Firefox. Same features, same shortcuts, same sync — across browsers.

Video controls

Play/pause, speed up/down, skip forward/back, mute, fullscreen — control any video on any site.

Reader mode

Strip ads and clutter from any page with one keypress. Just the content, beautifully formatted.

Smart copy

Copy page title, URL, or both as a markdown link. Copy all links on a page. Extract email addresses. Export tables as CSV.

Search & filter

Find any shortcut instantly by typing in the filter bar. Search by label, key, or action name.

Enable/disable toggle

Temporarily disable any shortcut or entire group without deleting it. One click to toggle.

Drag to reorder

Drag shortcuts to reorder within or between groups. Your order is saved.

Stats bar

See your total shortcuts, disabled count, and conflict count at a glance.

Cheat sheet overlay

Press a key to see all your active shortcuts overlaid on the current page. Press Escape to dismiss.

Insert text action

Type configured text into the focused field with a shortcut. Great for email templates, code snippets, signatures.

Toggle bookmark

Add or remove the current page from your bookmarks with a single keypress.

Tab superpowers

Close duplicates, sort alphabetically, suspend inactive tabs, group/ungroup, jump to audible tab, new tab to the right.

Open in incognito

Open the current page in a private window with one key. Quick, no clicking through menus.

QR code generator

Generate a QR code for the current page URL, displayed as a floating overlay you can scan from your phone.

Greasyfork import

Paste a Greasyfork or OpenUserJS URL and bind any userscript to a keyboard shortcut. No other extensions needed.

Developer tools

Show layout grid, clear site storage, performance metrics, pretty-print JSON, log all events — built right in.

Action feedback toasts

Subtle confirmations when copying, sorting, or toggling — so you always know your shortcut worked.

90+ Built-in Actions

Every action you can assign to a keyboard shortcut, organized by category.

Your shortcuts are safe

All your existing shortcuts were automatically migrated. Everything works just like before — plus everything above.
